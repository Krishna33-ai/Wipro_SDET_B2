JAVASCRIPT ASYNCHRONOUS PROGRAMMING - SYNOPSIS
===============================================

This script demonstrates three core async concepts in JavaScript:

1. PROMISE (getData function)
   - Simulates an async data fetch using a 2-second delay.
   - Resolves with "Data received" on success.
   - .then() handles success, .catch() handles errors.

2. setTimeout
   - Runs once after a 5-second delay.
   - Used for one-time delayed execution.

3. setInterval
   - Runs repeatedly every 3 seconds.
   - Used for continuous/polling tasks.

KEY TAKEAWAY:
Promises handle async operations cleanly with chaining,
while setTimeout and setInterval manage time-based execution
— together forming the backbone of non-blocking JavaScript.
